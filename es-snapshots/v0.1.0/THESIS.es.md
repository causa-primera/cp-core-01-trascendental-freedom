# cp-core-02 — Funcionalismo: Derrota interna (v0.1.0)

## 1. Tesis
Este repositorio sostiene que el **funcionalismo fuerte sobre la agencia** sufre una **derrota interna**: si la agencia se reduce a rol conductual/funcional, entonces las condiciones normativas necesarias para imputar responsabilidad (suspensión, evaluación, compromiso) quedan presupuesta sin fundamento o desplazadas a procedimientos externos.

## 2. “Derrota interna”
Una posición queda internamente derrotada cuando:
- necesita una distinción normativa para funcionar (p. ej., “compromiso asumido” vs “salida”),
- pero no puede justificar esa distinción con sus propios recursos (rol funcional), salvo importando primitivas normativas ajenas.

## 3. Blanco
No es el uso ingenieril de “se comportan parecido”, sino la tesis de equivalencia fuerte:
- si dos sistemas son funcionalmente equivalentes, lo son respecto de agencia/razones/responsabilidad.

## 4. Relevancia para Causa Primera
Causa Primera requiere una capacidad no delegable: suspensión genuina y compromiso atribuible.
Si el compromiso es un rol realizable por cualquier sistema, la “firma” se vuelve propiedad de pipelines y la responsabilidad se reasigna silenciosamente a Causas Segundas.

## 5. Puente con core-01 y core-03
- Core-01 aporta las primitivas S→E→K.
- Core-03 aporta las restricciones de aparato contra la sustitución silenciosa.
Core-02 aporta el resultado negativo: la equivalencia funcional no basta para asegurar apropiación normativa.
